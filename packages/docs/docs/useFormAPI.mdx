# useForm

### Function setForm()

Object properties that function receives:

| Property           |                                  Description                                  |
| ------------------ | :---------------------------------------------------------------------------: |
| `initialValues`    |     This optional property receives a object with initial values of form.     |
| `initialErrors`    |     This optional property receives a object with initial errors of form.     |
| `initialTouched`   | This optional property receives a object with initial touched fields of form. |
| `isControlled`     |        This optional property determines if the fields are controlled.        |
| `debounced`        |        This optional property determines if the fields are debounced.         |
| `schemaValidation` |           This optional property receives a you schema validation.            |

This function update the form state, when using this function it's possible to pass an object or a callback function, this function receives a state as param.
This aproach turn the form state update straightforward.

useForm receives a object with some optional properties, and setForm works the same way, it's posible to omit some properties when run setForm,
but the object received will replace the current state of form.

```javascript

   const newValues = {...}

   setForm( state => ({
      ...state,
      initialValues: newValues
   }))

   //.....or....//

   setForm({
      initialValues:newValues,
      initialErrors:{...},
      initiaalTouched:{...}
      ...
   })

```
